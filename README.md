PosMApp
=======

PosMAppは、ポスターセッション参加者向けのポスターマップアプリです。  
クリッカブルポスターマップやポスター検索機能、ブックマーク機能があるので、  
紙媒体やPDFで確認するというやり方とは違い、  
少ない手間で興味のあるポスターの位置を確認したり、投票したりすることができます。
